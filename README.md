# RigLog 🩸💪

RigLog is a personal health analytics desktop app built in Python.

The goal is to combine multiple health data sources into a single application for analysis and visualisation, including:

- Glucose data
- Activity data
- Workout data
- Nutrition data

## Tech Stack

- Python
- PySide6
- pandas
- SQLAlchemy
- SQLite

## Project Status

Phase 1 — Foundation complete

Current progress:
- Project structure created
- Python environment set up
- PySide6 installed
- SQLite database connected
- Basic multi-tab window built

## Roadmap

- Glucose import
- Glucose event classification
- Glucose line chart with target range
- Fitbit integration
- Workout logging
- Nutrition tracking

```bash
python -m app.main
```
